# 동의보감 스크래핑

본 Repository는 [한의학고전DB](https://mediclassics.kr/books/8/)라는 웹 사이트에서 동의보감을 시험삼아 Node.JS로 스크래핑해서 로컬에 텍스트 파일로 저장하는 소스코드입니다. 각 js 확장자 파일(common.js 제외)을 Node.JS로 실행시키면 동의보감 현대 한글 번역 전문이 들어있는 텍스트 파일이 생깁니다.
